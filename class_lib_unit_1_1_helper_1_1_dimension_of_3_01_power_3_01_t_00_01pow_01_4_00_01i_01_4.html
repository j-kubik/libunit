template<typename T, int pow, int i>
class LibUnit::Helper::DimensionOf< Power< T, pow >, i >

Helper class used to extract dimension form a type representing a unit.

Template Parameters
TUnit for which dimension is computed.
iFor internal use only. It has a default value and should never be explicitly specified.

Input type must be a proper unit ie. all simple units used must define Dimension subtype.

Specialization for power types.
