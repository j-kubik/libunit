Detailed Description

template<typename T, int i>
class LibUnit::Helper::Simplify< Compound< T >, i >

Helper class used for simplifying a type.

Template Parameters
Compound<T>Type to be simplified.
iFor internal use only. It has a default value and should never be explicitly specified.

Type that is to be simplified must be flat. Simplified type is a flat type that has at most one power of any singular type. If simplified type has only a power of one singular type, then it should be an instance of a Power class, not wrapped in a Compound. If in addition power of this singular type equals one, simplified type equals to this simple type, not wrapped in Power or Compound. Empty-unit (dimensionless) is represented by empty Compound<>.

Specialization for Compound types containing only one element.
