Detailed Description

template<typename From, typename To>
class LibUnit::Convert< From, To >

Template used to convert value expressed in one unit to value expressed in another.

Template Parameters
FromUnit in which input value is expressed.
ToUnit in which result value is expressed.
Remarks
You can specialize Convert class in order to allow for non-standard conversions. If you specialize Convert for a conversion that is not possible by default, you should also specialize Convertible, otherwise compilation error will ocur before any canoversion can take place.
