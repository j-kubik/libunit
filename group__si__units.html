Detailed Description

These are definitions of units included in SI unit system. Units and dimensions are exactly as in International System of Units, with exception of Gram, which is defined as basic instead of a kilogram. This doesn't introduce any practical differences in usage of those units in LibUnit.

Multiple SI prefixes can be applied to a unit to create a derived unit that differes only by a factor of integral power of ten. It should only be applied to units, but not to dimensions. Lib unit provides set of helper types representing such prefixes.

Examples

Kilo<A> // is a unit 1000x bigger than unit A.
Mili<A> // is a unit of 1/1000 of A.
Kilo<Metre> // a kilometre.
Kilo<Gramm> // a kilogram.
Giga<Metre> // a gigametre
Kilo<Kilo<Metre>> // also a gigametre
Mili<Ampere> // miliampere

SI units and prefixes are supplementary to LibUnit. There is no requirement to use SI units provided with LibUnit. If other set of units is more appropriate, it should be used instead.
